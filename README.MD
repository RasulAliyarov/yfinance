Выручка (динамика) — рост общего объема продаж (денег «в кассе») по сравнению с прошлым годом.

Прибыль (динамика) — рост чистой прибыли (денег «в кармане») после вычета всех расходов и налогов.

FCF ($) — свободный денежный поток (реальный кэш), оставшийся после всех трат на поддержание бизнеса.

P/E — коэффициент «Цена / Прибыль», показывающий, за сколько лет окупятся вложения в акцию.

Ликвидность (Current Ratio) — способность компании покрыть долги своими текущими активами (в идеале > 1.1).

Маржа (%) — эффективность бизнеса, показывающая, какой процент от выручки превращается в чистую прибыль.

P/FCF — рыночная цена компании относительно её свободного денежного потока (чем ниже, тем дешевле акция).

Акции в обращении — проверка наличия «байбэка» (выкупа акций компанией) или отсутствия размытия долей инвесторов.

Див/FCF (%) — какой процент от свободных денег уходит на дивиденды (безопасно, если меньше 70%).
